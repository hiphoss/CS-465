# CS-465 Travlr Getaways

This repository contains coursework for CS-465 Full Stack Development. The project is based on the Travlr Getaways web application and is developed incrementally across course modules.

The application uses a MEAN stack architecture, with Express and Node.js handling server-side logic and Handlebars used for server-rendered views. Additional functionality and architecture components are added throughout the course.

## Project Structure
- app_server: Server-side MVC structure including routes, controllers, and views
- public: Static assets such as stylesheets and images
- app.js: Express application entry point

## Branches
Each module is developed on its own branch to reflect incremental progress:
- module3: Dynamic rendering using JSON and Handlebars templates

Future modules will continue to build on this structure.

## Notes
This branch represents the baseline project structure. Specific module implementations can be found in their respective branches.
