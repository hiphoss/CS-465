# CS-465 Module Two – MVC Routing and Handlebars

## Overview
This module focused on refactoring a static Express application into a structured Model View Controller (MVC) architecture. The goal was to improve organization, maintainability, and prepare the application for future full stack development.

In this module, the Travlr Getaways website was converted from static HTML and default Express routing to a dynamic setup using controllers, routes, and Handlebars (HBS) templates.

---

## Key Concepts Applied
- Model View Controller (MVC) architecture
- Express routing and controllers
- Handlebars (HBS) templating engine
- Dynamic rendering of views
- Separation of application logic and presentation

---

## Implementation Details
- Created an `app_server` directory to contain MVC components
- Implemented controllers to handle request logic
- Defined routes to map URLs to controller actions
- Converted static pages into Handlebars templates
- Updated `app.js` to register HBS and MVC routes
- Verified functionality by rendering multiple routes successfully

---

## Technologies Used
- Node.js
- Express
- Handlebars (HBS)
- JavaScript
- Git and GitHub

## How to Run
1. Install dependencies:
   npm install
2. Start the application:
   npm start
